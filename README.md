# Word Count — подсчёт слов в тексте

## Описание задачи

Необходимо реализовать программу, которая анализирует текст и считает, сколько раз встречается каждое слово.

Например, для строки: `"cat dog cat bird dog cat"` результат: `cat -> 3`, `dog -> 2`, `bird -> 1`.

## Обязательная часть

- чтение строки или текста
- разбиение текста на слова
- подсчёт количества повторений каждого слова
- вывод результата в формате `слово -> количество`

## Вариативная часть

Из подсчёта исключаются служебные слова (стоп-слова): `и`, `в`, `на`, `под`, `за`, `с`, `the`, `a`, `an`, `and`, `in`, `on`

## Пример работы программы

**Вход:** `Cat, dog, and cat! Bird, dog, on the cat.`

**Результат:**
cat -> 3
dog -> 2
bird -> 1


## Как работает программа

1. Приводит текст к нижнему регистру
2. Разбивает текст на слова (через `re.findall`, игнорируя знаки препинания)
3. Исключает стоп-слова
4. Подсчитывает частоту каждого слова через словарь
5. Выводит результат
